## About The Project

In a world where time and reality fell apart, society now exists within isolationist driven factions based around different cultures, mythologies, and fantasy stories. Your interaction with a god gave you not only the ability to manipulate time, but also the purpose of restoring the world's original state. The world is centered around serving one's self interest, so who knows what facts you may uncover about the true causes of the chaos!  
  
This is an end-of-year game project for a Video Game Programming class in Frisco ISD. The goal of this project will be to produce a game that is good enough to spend $100 and publish it to Steam. All assets not made by our team of 3 will receive full credit and payment when possible.
